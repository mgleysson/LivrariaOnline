# LivrariaOnline

Sistema de Livraria Online

O referente sistema de Livraria Online é uma linha de produto de software que consiste em gerenciar a aquisição de livros feita pelos clientes que adicionam em seu carrinho de compras os livros desejados. No sistema deve haver o administrador do sistema que gerencia todo o estoque de livros da livraria, estes que pertencem a uma determinada categoria. Além disso, o carrinho de compras será preenchido com um conjunto de itens de compra que contém a quantidade de produtos juntamente com o valor unitário para que se possa ter ao fim o valor total da compra feita pelo cliente da livraria.
